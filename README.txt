In anaconda prompt, run the following commands //to install cv2 module

	- conda install pip

create a folder facedata in the same location as of the code files

run data_creator.py, Enter register number and name
the face data will be stored in the folder facedata  (C:\users\user_name\anaconda3\facedata in my laptop)

run trainer.py

run detector.py
a new window "my face" will be opened with the camera and detector will detect already existing members data
